1

Get the backup files from each phone

You need two files named msgstore.db.crypt15 — one from each phone.

  • Connect the phone to your PC via USB cable
  • On the phone: swipe down the notification bar → tap Charging via USB → select File Transfer
  • On your PC: open File Explorer and find the phone under This PC
  • Navigate to the backup folder:
📱 Android 11 or newer:
Internal Storage → Android → media → com.whatsapp → WhatsApp → Backups → msgstore.db.crypt15
or
📱 Android 10 or older:
Internal Storage → WhatsApp → Databases → msgstore.db.crypt15

Copy the file and rename it so you can tell them apart, e.g. old_backup.crypt15 and new_backup.crypt15.

💡Not sure which Android version? Open Settings → About phone → Android version.
2

Get the encryption key from each phone

Each backup is locked with a secret key. You need the key for each phone separately.

On the phone, open WhatsApp:

⚙️ Settings → Chats → Chat Backup → scroll to the bottom → Backup encryption key → Export

This exports a file called encrypted_backup.key. Copy it to your PC the same way as the backup file above.

or

If you see a 64-character code on screen instead of an Export button, copy all the characters and paste them into a plain .txt file — that works too.

⚠️The key for the old phone is different from the key for the new phone. Do not mix them up.
💡"Backup encryption key" only appears if you have end-to-end encrypted backups turned on. If you don't see it, go to Chat Backup → End-to-end Encrypted Backup → Turn on, then come back.
3

Upload the 4 files below

Left column = your old phone. Right column = your new phone. Each column needs its backup file and its key file. That's it — click Start Merge and wait.

💡You can drag files straight from File Explorer and drop them onto the boxes below.
4

Restore the merged backup on your new phone

After the merge finishes, you'll get a Merged Backup file and a Restore Guide to download. The full steps are in that guide, but here's a quick summary:

  • Uninstall WhatsApp from your new phone
  • Connect the new phone to your PC via USB (File Transfer mode)
  • Copy msgstore.db.crypt15 to the backup folder on the new phone:
Internal Storage → Android → media → com.whatsapp → WhatsApp → Backups
  • Re-install WhatsApp and verify your number
  • When asked to restore a backup — tap Restore
  • Your full chat history (both phones merged) will appear
💡Copy your old photos and videos too — see the Media Summary download for exact folder paths.
⚠️Don't open WhatsApp until after you've placed the backup file — otherwise WhatsApp creates a fresh empty database and you'll lose the merge.
